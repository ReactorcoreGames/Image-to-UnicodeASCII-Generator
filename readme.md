# Image to Unicode/ASCII Generator

## About
Drag & drop an image (jpg, png, or gif) into the web app and it will generate a Unicode/ASCII version of the image, respecting its original aspect ratio. 

You can tweak the size and if it should compensate for letter symbols being taller than wider.
